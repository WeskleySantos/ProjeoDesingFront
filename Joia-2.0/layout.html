Aula sobre Layouts - FLEX / FLOAT

Lembrar que: com flex se usa o padding e a margin em % (não é recomendado, mas é o jeito)
No float se usa px para padding e margin

A aula de layouts é sobre flex e float, uma forma mais fácil de definir o layout de uma página com CSS. Lembrar de colocar div container no projeto orientado. Box-sizing é algo novo que está sendo ensinado, inclusive border-box
